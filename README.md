# 🗡️ Quantum Chain — Kamus Sambung Kata (Roblox)

Aplikasi pendamping untuk game **Sambung Kata** di Roblox. Cari kata berdasarkan
**awalan**, **akhiran**, **suffix-spam berlapis**, dan **word trap**, lengkap
dengan asisten strategi, mode overlay, dan tema (gelap / terang / sakura).

> **Kamus ini tidak diperjualbelikan.** Token hanya bisa didapatkan dari:
> **Roblox:** `blondesigma666`  ·  **Discord:** `caraphelina666`

---

## ⬇️ Cara Unduh & Pakai

1. Klik halaman **[Releases](../../releases)** (sisi kanan, atau tab "Releases").
2. Pada rilis paling baru, unduh file **`QuantumChain.exe`**.
3. Taruh di satu folder khusus (mis. `Documents\QuantumChain\`), lalu **klik dua
   kali** untuk menjalankan — **tanpa install**.
4. Masukkan **token** yang kamu dapat (lihat kontak di atas) untuk login.

---

## 📌 Catatan Penting

- Saat pertama dijalankan, folder `data/` & `backup/` dibuat otomatis **di samping
  `.exe`** — di situlah progres kamu tersimpan.
- Saat ada **update**, ganti `.exe` lama dengan yang baru **di folder yang sama**,
  dan **JANGAN hapus folder `data/`** (itu progres index kamu).
- **Launch pertama** bisa agak lambat (wajar). Bila **antivirus** menahan/minta
  izin, izinkan saja — ini umum untuk aplikasi semacam ini.
