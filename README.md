Quantum Chain membantumu menemukan kata dengan **cepat** saat bermain **Sambung Kata** — cari kata berdasarkan **awalan/akhiran**, deteksi **jebakan**, kumpulkan kosakata, dan pantau progres harianmu. Tampilan custom yang ringan, plus mode pertandingan.

## ✨ Fitur Utama

- 🔍 **4 mode pencarian** — Awalan, Akhiran, Awalan+Akhiran, dan Spam Akhiran.
- 💀 **Deteksi jebakan** — skor jebakan bertingkat (sampai **LETHAL**), Cari Jebakan, Jebakan Berantai, & Peta Rantai huruf.
- 📚 **Kamus Cadangan (KBBI)** — ribuan kata tambahan muncul otomatis saat koleksimu habis.
- 🗃️ **Indeks & farming kosakata** — kumpulkan kata ke "vault"-mu, lihat **Saran Farm**, target harian, dan rekor.
- 📥 **Tabel "Sudah Dicoba"** — singkirkan kata yang tak terpakai **tanpa** menghapusnya permanen.
- 🏆 **Mode Pertandingan** — timer match, hitung kata, dan **Analisa Match** (rekap & statistik pertandinganmu).
- 🔔 **Notifikasi toast** — perayaan (target, rekor, LETHAL, menang) tampil ringkas, tidak mengganggu.
- 🧰 **Alat bantu** — Command Palette (`Ctrl+K`), mode Overlay (selalu di atas), dan banyak profil.
- 🎨 **3 tema** — Dark, Light, Sakura.
- 📱 **Responsif** — nyaman dipakai di layar kecil & split-screen.

## 🔍 Mode Pencarian

| Mode | Fungsi |
|---|---|
| **Awalan** | Cari kata yang diawali huruf/penggalan tertentu |
| **Akhiran** | Cari kata yang diakhiri huruf/penggalan tertentu |
| **Awalan + Akhiran** | Cari kata dengan awalan **dan** akhiran sekaligus |
| **Spam Akhiran** | Cari kata "skakmat" — counter lawan paling sedikit di atas |

## 📥 Unduh & Jalankan

1. Buka **[halaman Rilis](https://github.com/caraphelina/quantumchain-download/releases/latest)**.
2. Unduh **`QuantumChain.zip`** dari rilis terbaru.
3. **Ekstrak** folder-nya (jangan dijalankan dari dalam zip).
4. Buka **`QuantumChain.exe`**.

> 💻 Windows · tanpa instalasi — cukup ekstrak lalu jalankan.

## ⌨️ Pintasan Penting

| Pintasan | Fungsi |
|---|---|
| `Ctrl+K` | **Command Palette** (semua perintah) |
| `Alt+A` / `Alt+S` | Mulai / hentikan match |
| `Alt+Q` / `Alt+E` | Fokus input Awalan / Akhiran |
| `Alt+W` | Tukar Awalan ↔ Akhiran |
| `Alt+Z` | Urutkan terpanjang / terpendek / default |
| `Alt+C` | Spam Chain Mode |
| `Alt+M` | Peta Rantai huruf |
| `Alt+T` | Mode Overlay (selalu di atas) |
| `Alt+[` / `Alt+]` | Perkecil / perbesar font |
| `←` / `→` | Geser huruf **akhiran** *(mode Awalan+Akhiran)* |
| `Ctrl+←` / `Ctrl+→` | Geser huruf **awalan** *(mode Awalan+Akhiran)* |
| `Shift+Home` / `Shift+klik-kanan` | Singkirkan kata → **Sudah Dicoba** |

*Daftar lengkap perintah ada di Command Palette (`Ctrl+K`).*
